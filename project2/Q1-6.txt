The relevant security property is forward secrecy. Even after Mallory 
compromises Alice’s device, she cannot decrypt earlier messages because the 
Double Ratchet deletes old message keys and advances the chain key forward. Once
ratcheted, past keys cannot be derived again, so previous messages remain 
confidential.

6. 
(a)
RSA keys take much longer to generate.
RSA key generation requires generating large random primes (4096-bit in our 
test), which is computationally expensive. In contrast, ECDSA key generation 
only requires choosing a random scalar in the group, which is significantly 
faster.

(b)
RSA signatures take longer to generate 
RSA signing involves a large modular exponentiation with a 4096-bit private key.
ECDSA signing uses group operations that are faster at comparable security 
levels.

(c) 
RSA signatures are longer
With 4096-bit RSA, signatures are 512 bytes, whereas ECDSA P-384 signatures are
only 96 bytes.

(d)
ECDSA verification takes longer
RSA verification is faster because it uses a small public exponent, while ECDSA
verification involves more expensive group operations.
